Trabalho de Desenvolvimento de Software

Este projeto tem como objetivo a criação de uma página web estruturada em HTML, estilizada com CSS e com funcionalidades dinâmicas implementadas em JavaScript. A página apresenta um cabeçalho com o nome do projeto e um menu de navegação, seguido por uma seção dedicada aos integrantes da equipe, contendo fotos, descrições e links para redes sociais. Além disso, há uma tabela detalhando as contribuições de cada membro, bem como uma seção explicativa sobre o projeto e as tecnologias utilizadas. Por fim, um rodapé é adicionado para incluir contatos e créditos do desenvolvimento. O design é aprimorado com cores, fontes personalizadas e espaçamentos adequados, garantindo uma interface visualmente agradável e organizada. A interação dinâmica da página, implementada via JavaScript, adiciona um elemento de interatividade, tornando a experiência do usuário mais envolvente.
